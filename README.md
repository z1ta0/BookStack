# BookStack

BookStack is a 3D spine-first book stack app for tracking your reading with a clean, tangible visual, using Open Library cover art for realistic spines and saving data locally in the browser.

![BookStack Preview](.github/preview.png)

## Quick Start

1. Install dependencies
   ```bash
   npm install
   ```
2. Start dev server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Tech Stack

- Vanilla JS (ES modules), HTML, CSS
- Vite
- Open Library API
